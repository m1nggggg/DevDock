# DevDock Product Context

## Register
product

## Purpose
DevDock is a small internal launchpad that collects developer tool links in one clean place. It should make common tools easy to find and open without remembering separate URLs.

## Users
Developers and delivery team members who need quick access to project tools during daily work.

## Current Tools
- Planning Poker
- CSV Splitter
- Realm Manager
- Menu Visualizer

## Product Principles
- Keep navigation simple while the tool list is small.
- Prefer search and clear labels over complex categories.
- Tool data should be separated from UI code so links can be updated without changing components.
- The experience should feel polished, fast, and lightweight.

## Tone
Calm, useful, modern, and direct. Avoid enterprise clutter.
